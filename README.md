# Oropeza-Post2-U7
API REST Productos — Spring Boot

Este proyecto consiste en la implementación de una API REST completa con Spring Boot utilizando 
@RestController y ResponseEntity para exponer endpoints CRUD sobre una colección de productos en memoria, aplicando los códigos HTTP correctos para cada operación.

ruta de ejecucion:
1. Clonar el repositorio
2. Abrir el proyecto en IntelliJ IDEA
3. Ejecutar con Maven
4. La API estará disponible en http://localhost:8080
